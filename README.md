# 🎮 Monte Carlo AI Game Simulator

## 📌 Overview
The **Monte Carlo AI Game Simulator** is an interactive web-based project that demonstrates how probabilistic AI can be used for decision-making in games. The system uses **Monte Carlo simulation techniques** to evaluate thousands of possible future outcomes and selects the most optimal move based on probability analysis.

This project combines **Artificial Intelligence, Game Theory, and Interactive UI Design** to create an engaging and intelligent gameplay experience.

---

## 🚀 Features

- 🎮 Player vs AI mode with intelligent decision-making  
- 👥 Player vs Player mode for local multiplayer gameplay  
- 🤖 Monte Carlo simulation-based AI engine  
- 🎯 Multiple difficulty levels (Easy / Medium / Hard)  
- 🟡 AI Heatmap visualization for move probability analysis  
- 📊 Real-time scoreboard tracking wins and results  
- 🌙 Dark mode / Light mode UI toggle  
- 🔊 Sound effects for interactive feedback  
- ✨ Winner highlight animation for visual results  

---

## 🧠 How the AI Works

The AI uses **Monte Carlo simulation** to:

1. Simulate multiple random future game states  
2. Evaluate possible outcomes for each move  
3. Count win/loss probabilities  
4. Select the move with the highest success rate  

This allows the AI to make **data-driven decisions instead of fixed rule-based moves**.
